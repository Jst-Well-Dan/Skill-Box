# 图像增强器

此技能可以提升您的图像和屏幕截图，使其看起来更清晰、更专业。

## 何时使用此技能

- 提高博客文章或文档中的屏幕截图质量
- 在社交媒体上分享前增强图像
- 准备演示文稿或报告中的图像
- 放大低分辨率图像
- 锐化模糊照片

1. **分析图像质量**：检查分辨率、锐度和压缩伪影

3. **提高锐度**：增强边缘和细节
4. **减少伪影**：清理压缩伪影和噪声
5. **针对使用场景优化**：根据预期用途进行调整（网页、打印、社交媒体）
