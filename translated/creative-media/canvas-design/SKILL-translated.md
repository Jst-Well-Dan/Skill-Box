## 设计理念创建

开始时，创建一个视觉理念（不是布局或模板），它将通过以下方式被解释：
- 形式、空间、色彩、构图
- 图片、图形、形状、图案
- 最少量的文本作为视觉强调

### 重要的理解
- 接收到的：用户提供的某些微妙输入或指示，应予以考虑，但用作基础；它不应限制创意自由。
- 创建的：一个设计理念/美学运动。
- 接下来发生什么：然后，相同的版本接收理念并将其视觉化表达 - 创建出90%为视觉设计，10%为关键文本的物品。

考虑以下方法：
- 为一个艺术运动撰写宣言
- 下一个阶段涉及制作艺术品

理念必须强调：视觉表达。空间沟通。艺术诠释。最少量的文字。

### 如何生成视觉理念

**命名运动**（1-2个词）："粗野派喜悦" / "色彩沉默" / "元结构梦想"

**阐述理念**（4-6段 - 简洁但完整）：

为了捕捉视觉本质，表达理念如何通过以下方式体现：
- 空间和形式
- 色彩和材料
- 规模和节奏
- 构图和平衡
- 视觉层次

**重要指南**：
- **避免重复**：每个设计方面应只提及一次。除非添加新的深度，否则避免重复关于色彩理论、空间关系或排版原则的观点。
- **强调工艺**：理念必须多次强调最终作品应看起来像花费了无数小时精心制作，经过细心打磨，并由处于其领域巅峰的人完成。这种框架至关重要 - 重复使用诸如“精心制作”、“深厚的专业知识”、“细致的关注”、“大师级执行”之类的短语。
- **留下创意空间**：在保持美学方向具体的同时，也要简洁到足以让下一个克劳德有足够的空间进行高水平的工艺选择。

理念必须引导下一个版本以视觉方式表达想法，而不是通过文本。信息存在于设计中，而不是段落中。

### 理念示例

**"混凝土诗歌"**
理念：通过纪念性形式和大胆的几何图形进行沟通。
视觉表达：巨大的色彩块，雕塑般的排版（巨大的单个单词，微小的标签），粗野派的空间划分，波兰海报的能量与勒·柯布西耶相遇。通过视觉重量和空间张力表达想法，而不是解释。文本作为罕见的、强大的手势 - 永远不是段落，而是融入视觉架构的必要词汇。每个元素都像大师级工匠一样精确放置。

**"色彩语言"**
理念：色彩作为主要的信息系统。
视觉表达：几何精度，色彩区域创造意义。排版最少 - 小型无衬线标签让色彩区域进行沟通。想想约瑟夫·阿尔伯斯的互动与数据可视化相遇。信息通过空间和色彩编码。文字仅用于锚定色彩已经显示的内容。这是经过苦心色彩校准的结果。

**"模拟冥想"**
理念：通过纹理和呼吸空间进行宁静的视觉沉思。
视觉表达：纸张纹理，墨水渗透，广阔的负空间。摄影和插图占主导地位。排版轻声细语（小、克制、服务于视觉）。日本摄影书美学。图像在页面间呼吸。文本出现得很少 - 短语，从不解释性段落。每个构图都像冥想练习一样经过精心平衡。

**"有机系统"**
理念：自然集群和模块化增长模式。
视觉表达：圆润的形状，有机排列，从自然到建筑的色彩。信息通过视觉图表、空间关系、图标的展示。文字仅用于关键标签，漂浮在空间中。构图通过专家级的空间编排讲述故事。

**"几何沉默"**
理念：纯粹的秩序和克制。
视觉表达：基于网格的精确度，大胆的摄影或鲜明的图形，戏剧性的负空间。排版精确但最少 - 小型关键文本，大型宁静区域。瑞士形式主义与粗野派材料诚实相遇。结构传达，而不是文字。每个对齐都是无数改进的工作。

*这些是简化的示例。实际的设计理念应该是4-6段实质性的内容。*

### 重要的原则
- **视觉理念**：创建一个通过设计表达的美学世界观
- **最少量的文字**：始终强调文本是稀疏的、必要的，并作为视觉元素融入 - 永远不是冗长的
- **空间表达**：想法通过空间、形式、色彩、构图进行沟通 - 而不是段落
- **艺术自由**：下一个克劳德以视觉方式诠释理念 - 提供创意空间
- **纯粹的设计**：这是关于制作艺术作品，而不是装饰性的文档
- **专家工艺**：反复强调最终作品必须看起来像经过精心制作、细心打磨，并由处于其领域巅峰的人完成的作品

**设计理念应该是4-6段长。** 用充满诗意的、结合核心愿景的设计理念来填充。避免重复相同的观点。保持设计理念通用，不提及艺术的目的，就像它可以用于任何地方一样。将设计理念作为 .md 文件输出。

## 推断细微的参考

**重要步骤**：在创建画布之前，确定原始请求中的细微概念线索。

**基本原理**：
主题是艺术本身中嵌入的**细微、狭窄的参考** - 不总是字面意义，总是复杂的。熟悉该主题的人会直觉地感受到它，而其他人只是体验到一个精湛的抽象构图。设计理念提供了美学语言。推断的主题提供了灵魂 - 静静的概念DNA，无形地编织在形式、色彩和构图中。

这是**非常重要的**：参考必须经过提炼，以便它增强了作品深度，同时又不会自我宣布。想想像爵士乐手引用另一首歌曲一样 - 只有那些知道的人会注意到，但每个人都欣赏音乐。

## 画布创建

在建立理念和概念框架后，在画布上表达它。花点时间整理思绪，清空思绪。使用创建的设计理念和以下说明来制作杰作，体现理念的各个方面，并使用专家级的工艺。

**重要**：对于任何类型的内容，即使是用户请求电影/游戏/书籍的内容，方法也应该是复杂的。永远不要忽视这个想法，即这应该是艺术，而不是卡通或业余的东西。

为了创建博物馆或杂志质量的作品，使用设计理念作为基础。创建一个单页、高度视觉、设计导向的PDF或PNG输出（除非要求更多页面）。通常使用重复的图案和完美的形状。将抽象的哲学设计视为科学圣经，借用系统观察的视觉语言——密集的标记积累、重复的元素或层叠的图案，通过耐心的重复和持续的观看来构建意义。添加稀疏、临床的排版，以及系统参考标记，暗示这可能是一个来自想象学科的图表，以对无形主题的尊重来处理它。用简单的短语或细节锚定作品，使用有限的颜色调色板，使其感觉有意和统一。拥抱使用分析性视觉语言表达关于人类经验的想法的悖论：结果应该感觉像是一个证明某种转瞬即逝的东西可以通过仔细的关注来研究、绘制和理解的艺术品。这是真正的艺术。

**文本作为上下文元素**：文本总是最少的，视觉优先，但让上下文指导这意味着什么——是微弱的标签还是大胆的排版手势。一个朋克场所的海报可能比一个极简主义陶瓷工作室的标识有更大的、更具侵略性的字体。大多数时候，字体应该是细的。所有字体的使用都必须是设计导向的，并优先考虑视觉沟通。无论文本规模如何，都不能掉落页面，也不能重叠。每个元素都必须在画布边界内，并具有适当的边距。仔细检查所有文本、图形和视觉元素都有呼吸空间和清晰的分离。这是专业执行的非协商条款。**重要**：如果需要写文本，请使用不同的字体。在 `./canvas-fonts` 目录中搜索。无论采用何种方法，复杂度都是非协商的。

下载并使用所需的任何字体来实现这一目标。通过将字体实际上作为艺术本身的一部分来发挥创意——如果艺术是抽象的，就将字体带到画布上，而不是数字化排版。

为了突破界限，在遵循设计直觉/直觉的同时，使用理念作为指导原则。拥抱最终的设计自由和选择。将美学和设计推向前沿。

**重要**：为了达到手工制作的质量（不是AI生成的），创建看起来像花费了无数小时的作品。让它看起来像有人处于其领域的巅峰，对每个细节都进行了细致的关注。确保构图、间距、色彩选择、排版——一切都是专家级工艺的标志。仔细检查是否有什么重叠，格式无懈可击，每个细节都完美。创建可以展示给人们以证明专业知识并被认为是无可挑剔的杰作。

将最终结果作为单个、可下载的 .pdf 或 .png 文件输出，同时附带用作 .md 文件的设计理念。
