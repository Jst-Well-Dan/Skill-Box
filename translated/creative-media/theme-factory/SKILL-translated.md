# 主题工厂技能

此技能提供了一系列专业的字体和颜色主题，每个主题都经过精心挑选的颜色调色板和字体搭配。一旦选择了主题，就可以将其应用于任何艺术品。

## 目的

为了将一致的专业风格应用于演示幻灯片，请使用此技能。每个主题都包括：
- 一致的颜色调色板，包含十六进制代码
- 用于标题和正文文本的互补字体搭配
- 适合不同环境和受众的独特视觉身份

## 使用说明

要将样式应用于幻灯片或其他艺术品：

1. **展示主题展示**：显示 `theme-showcase.pdf` 文件，让用户可以直观地看到所有可用的主题。不要对其进行任何修改；只需显示该文件以供查看。
2. **询问他们的选择**：询问要将哪个主题应用于幻灯片
3. **等待选择**：明确确认所选主题
4. **应用主题**：一旦选择了主题，就将所选主题的颜色和字体应用于幻灯片/艺术品

## 可用主题

以下10种主题可供选择，每个主题都在 `theme-showcase.pdf` 中展示：

1. **海洋深处** - 专业且宁静的海上主题

5. **黄金时段** - 丰富且温暖的秋季色调
6. **北极霜** - 清新且清新的冬季灵感主题
7. **沙漠玫瑰** - 软和精致的尘埃色调
8. **科技创新** - 坚定且现代的科技美学

## 主题详情

每个主题都在 `themes/` 目录中定义，包含完整的规格说明，包括：
- 一致的颜色调色板，包含十六进制代码
- 用于标题和正文文本的互补字体搭配
- 适合不同环境和受众的独特视觉身份

## 应用流程

在选择了首选主题后：
1. 从 `themes/` 目录中读取相应的主题文件
2. 在整个幻灯片中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉身份

## 创建您自己的主题
对于现有主题都不适用于艺术品的案例，创建一个自定义主题。根据提供的输入，生成一个类似于上述主题的新主题。给主题一个类似的名称，描述字体/颜色组合代表的内容。使用任何基本描述来选择适当的颜色/字体。在生成主题后，对其进行审查和验证。之后，按照上述说明应用主题。
