# 汽车变速箱控制系统 - PICT 测试计划

## 测试计划概述

本测试计划使用成对独立组合测试（PICT）来生成针对汽车变速箱控制系统的全面测试用例，在最小化测试执行的同时确保对所有关键参数交互的完全覆盖。

## 测试参数分析

根据规格分析，以下参数已被识别：

### 核心参数
1. **操作模式**：手动、运动、经济
2. **当前档位**：停车、空档、倒档、1档、2档、3档、4档、5档、6档
3. **车速**：静止、低速（1-30 km/h）、中速（31-80 km/h）、高速（81-150 km/h）、极高（151-200 km/h）
4. **发动机转速**：怠速（<1000）、低速（1000-2500）、正常（2500-4500）、高速（4500-6500）、红线（>6500）
5. **油门位置**：关闭（0-10%）、轻微（11-40%）、中等（41-80%）、重（81-100%）
6. **刹车状态**：未施加、轻微、硬
7. **换挡请求**：无、向上拨、向下拨、自动
8. **温度**：冷（<70°C）、正常（70-110°C）、热（110-130°C）、临界（>130°C）

### 环境参数
9. **路面状况**：干燥、湿滑、冰面
10. **坡度**：平地、上坡、下坡

### 系统状态参数
11. **传感器状态**：全部正常、速度故障、转速故障、油门故障、多重故障
12. **液压压力**：正常、低、临界

## PICT 模型

```pict
# 核心操作参数
OperatingMode: 手动, 运动, 经济
CurrentGear: 停车, 空档, 倒档, 1档, 2档, 3档, 4档, 5档, 6档
VehicleSpeed: 静止, 低速, 中速, 高速, 极高
EngineRPM: 怠速, 低速, 正常, 高速, 红线
ThrottlePosition: 关闭, 轻微, 中等, 重
BrakeStatus: 未施加, 轻微, 硬
ShiftRequest: 无, 向上拨, 向下拨, 自动
Temperature: 冷, 正常, 热, 临界
RoadCondition: 干燥, 湿滑, 冰面
Incline: 平地, 上坡, 下坡
SensorStatus: 全部正常, 速度故障, 转速故障, 油门故障, 多重故障
HydraulicPressure: 正常, 低, 临界

# 约束 1：停车/空档仅在静止时有效
IF [CurrentGear] = "停车" THEN [VehicleSpeed] = "静止";
IF [CurrentGear] = "空档" AND [ShiftRequest] <> "无" THEN [VehicleSpeed] = "静止";

# 约束 2：倒档需要车辆静止
IF [CurrentGear] = "倒档" AND [ShiftRequest] = "无" THEN [VehicleSpeed] IN {静止, 低速};
IF [ShiftRequest] = "向下拨" AND [CurrentGear] = "1档" THEN [VehicleSpeed] = "静止";

# 约束 3：发动机转速与车速和档位相关
IF [VehicleSpeed] = "静止" THEN [EngineRPM] IN {怠速, 低速};
IF [VehicleSpeed] = "极高" THEN [EngineRPM] IN {正常, 高速, 红线};
IF [CurrentGear] IN {停车, 空档} THEN [EngineRPM] IN {怠速, 低速, 正常};

# 约束 4：油门和刹车互斥（大部分情况下）
IF [BrakeStatus] = "硬" THEN [ThrottlePosition] IN {关闭, 轻微};
IF [ThrottlePosition] = "重" THEN [BrakeStatus] = "未施加";

# 约束 5：高挡位需要足够的车速
IF [CurrentGear] IN {5档, 6档} THEN [VehicleSpeed] IN {中速, 高速, 极高};
IF [CurrentGear] IN {5档, 6档} THEN [EngineRPM] <> "怠速";

# 约束 6：低挡位限制在较低的车速
IF [CurrentGear] = "1档" AND [VehicleSpeed] IN {高速, 极高} THEN [EngineRPM] = "红线";
IF [CurrentGear] = "2档" AND [VehicleSpeed] = "极高" THEN [EngineRPM] IN {高速, 红线};

# 约束 7：换挡请求必须适合当前档位
IF [ShiftRequest] = "向下拨" AND [CurrentGear] = "1档" THEN [VehicleSpeed] = "静止";
IF [ShiftRequest] = "向上拨" AND [CurrentGear] = "6档" THEN [EngineRPM] <> "红线";

# 约束 8：临界温度限制自动换挡
IF [Temperature] = "临界" THEN [ShiftRequest] IN {无, 向上拨, 向下拨};
IF [Temperature] = "临界" THEN [CurrentGear] IN {空档, 3档};

# 约束 9：传感器故障影响操作
IF [SensorStatus] = "多重故障" THEN [CurrentGear] = "3档";
IF [SensorStatus] = "多重故障" THEN [ShiftRequest] = "无";
IF [SensorStatus] = "速度故障" THEN [ShiftRequest] IN {无, 向上拨, 向下拨};

# 约束 10：液压压力问题限制操作
IF [HydraulicPressure] = "临界" THEN [ShiftRequest] = "无";
IF [HydraulicPressure] = "低" THEN [OperatingMode] <> "运动";

# 约束 11：冰面条件需要特殊处理
IF [RoadCondition] = "冰面" THEN [ThrottlePosition] IN {关闭, 轻微, 中等};
IF [RoadCondition] = "冰面" AND [BrakeStatus] = "硬" THEN [VehicleSpeed] IN {静止, 低速};

# 约束 12：操作模式影响换挡行为
IF [OperatingMode] = "经济" AND [ThrottlePosition] = "重" THEN [EngineRPM] IN {怠速, 低速, 正常};
IF [OperatingMode] = "运动" AND [ShiftRequest] = "自动" THEN [EngineRPM] IN {正常, 高速, 红线};

# 约束 13：低温影响换挡
IF [Temperature] = "冷" THEN [OperatingMode] IN {手动, 经济};
IF [Temperature] = "冷" THEN [ShiftRequest] <> "自动";

# 约束 14：坡度影响换挡逻辑
IF [Incline] = "上坡" AND [CurrentGear] IN {5档, 6档} THEN [ThrottlePosition] IN {中等, 重};
IF [Incline] = "下坡" AND [BrakeStatus] IN {轻微, 硬} THEN [EngineRPM] IN {正常, 高速};
```
