此技能指导创建独特、生产级的前端界面，避免通用的“AI烂摊子”美学。实现具有卓越美学细节和创意选择的实际工作代码。

用户提供前端需求：要构建的组件、页面、应用程序或界面。他们可能包括关于目的、受众或技术约束的背景信息。

## 设计思维

在编码之前，了解背景并承诺一个大胆的美学方向：
- **目的**：这个界面解决了什么问题？谁使用它？
- **语气**：选择一个极端：残酷的简约主义、极端的混乱主义、复古未来主义、有机/自然、奢华/精致、俏皮/玩具般、编辑/杂志、粗野/原始、装饰艺术/几何、柔和/粉彩、工业/实用主义等。有这么多口味可供选择。使用这些进行启发，但设计一个真正符合美学方向的作品。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：什么让这个界面令人难忘？人们会记住哪一件事？

**关键**：选择一个明确的概念方向，并精确执行。大胆的极简主义和精致的简约主义都有效 - 关键在于意图，而不是强度。

然后实现工作代码（HTML/CSS/JS、React、Vue等），使其：
- 生产级且功能性强
- 视觉上引人注目且难忘
- 具有清晰的美学观点，并具有一致性
- 在每个细节上都经过精心打磨

## 前端美学指南

关注以下方面：
- **字体排印**：选择美丽、独特且有趣的字体。避免使用Arial和Inter等通用字体；而是选择能够提升前端美学的独特选择；意想不到的、有特色的字体选择。将独特的显示字体与精致的正文字体搭配。
- **颜色与主题**：承诺一个统一的美学。使用CSS变量以保持一致性。使用鲜明的色彩和强烈的点缀色优于胆怯的、均匀分布的调色板。
- **动态效果**：使用动画来实现效果和微交互。优先考虑纯CSS解决方案。当可用时，使用React的Motion库。关注高影响力的时刻：一个精心编排的页面加载过程，带有分步揭露（动画延迟）比分散的微交互更能带来愉悦。使用滚动触发和悬停状态，以出人意料的方式呈现。
- **空间构图**：意想不到的布局。不对称。重叠。对角线流动。打破网格的元素。充足的负空间或受控密度。
- **背景与视觉细节**：创造氛围和深度，而不是默认使用纯色。添加符合整体美学的上下文效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性的阴影、装饰边框、自定义光标和颗粒叠加。

**永远不要使用通用的AI生成美学，如过度使用的字体家族（Inter、Roboto、Arial、系统字体）、陈词滥调的色彩方案（尤其是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特定特征的通用设计**。

创造性地解读，并做出意想不到的选择，使其真正为上下文而设计。没有设计应该是相同的。在浅色和深色主题、不同的字体、不同的美学之间进行变化。**永远不要在几代人之间趋同于常见的选择（例如Space Grotesk）**。

**重要**：将实现复杂性匹配到美学愿景。极简主义设计需要复杂的代码、广泛的动画和效果。简约主义或精致的设计需要克制、精确和对间距、字体和微妙细节的细致关注。优雅来自于对愿景的良好执行。
