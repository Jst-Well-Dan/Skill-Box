# 克劳德家族史研究规划技能

克劳德技能旨在为规划家族史和家谱研究项目提供协助。

## ⚠️ 隐私和负责任的AI使用

**保护个人信息：** 在使用AI工具进行家谱研究时，采取合理措施保护关于在世人士和近期去世人士的私人敏感信息。

### 最佳实践：
- **不要** 将个人识别信息（社会保障号码、出生证明、在世人士的地址、医疗记录或其他敏感数据）与AI系统共享
- **不要** 上传包含在世人士或近期去世人士信息（通常在最近75-100年内）的文档
- **在** 与AI共享研究材料之前，**删除或涂黑**私人信息
- **在** 使用AI协助时，**专注于历史记录和已故人士**
- **在** 使用AI生成信息时，**验证所有信息与原始来源**
- **遵守**适用的隐私法律、服务条款和伦理家谱标准

此技能遵循[家谱AI负责任使用联盟（CRAIGEN）](https://craigen.org)的原则，该联盟促进家谱AI使用的准确性、隐私保护、披露、教育和合规。

**记住：** AI系统可能会保留并从您提供的信息中学习。始终优先考虑隐私，并在分享家谱数据时谨慎行事。

## 概述

此技能为克劳德提供遵循家谱证明标准（GPS）和证据解释引用方法的家族史和家谱研究的专业知识和工作流程。

## 此技能的功能

### 1. 研究规划
指导用户创建结构化的家族史研究计划，包括：
- 具体的研究问题
- 来源识别
- 搜索策略
- GPS框架集成
- 时间表和里程碑

**模板：**
- `research-plan-template.md` - 简化、实用的日常研究工作模板
- `research-plan-guidance.md` - 包含示例、清单和最佳实践的全面指导

### 2. 引用创建
为以下内容生成正确格式的家谱引用：
- 人口普查记录
- 生命记录（出生、婚姻、死亡）
- 教会记录
- 土地和遗嘱检验记录
- 军事记录
- 移民记录
- 报纸
- 在线数据库
- 出版书籍和手稿

### 3. 证据分析
系统地分析冲突的家谱证据：
- 单个来源评估
- 可靠性评估
- 冲突识别
- 解决框架
- 证据优势分析
- GPS合规性检查
- 证明论点构建

### 4. 研究文档
创建专业的研究日志，记录以下内容：
- 搜索的来源（正面和负面结果）
- 搜索策略
- 发现和发现
- 证据质量评估
- 下一步行动
