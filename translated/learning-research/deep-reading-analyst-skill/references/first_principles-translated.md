# 基本原理思考

## 核心概念

剥去假设和惯例，达到基本真理，然后从头开始重建理解。

## 三步流程

### 第一步：识别假设

**问：** "我们对这个有什么假设是正确的？"

列出内容中的所有隐性和显性假设：
- 未陈述的前提
- 继承的智慧（“众所周知…”）
- 行业惯例
- 作为普遍法则处理的历史先例

**商业领域的例子：**
- 假设：我们需要一个实体店来销售产品
- 问题：这是根本正确的还是历史上有利可图的？

### 第二步：分解到基本原理

**问：** "什么是肯定、可证明的真理？"

减少到无可争议的事实：
- 物理定律（如果适用）
- 数学真理
- 核心人类需求/行为
- 你可以验证的因果关系

**分类框架：**
```
主张 → 问：“我怎么知道这是真的？”
    ↓
├─ “它就是如此” → 很可能是假设，深入挖掘
├─ “一直都是这样做的” → 惯例，不是基本原理
├─ “X发生，然后Y发生” → 可能是基本原理（验证因果关系）
└─ “数学上/物理上必须是真的” → 基本原理
```

### 第三步：从头重建

**问：** "仅从基本原理出发，什么逻辑上随之而来？"

不引入旧假设重建：
1. 从经过验证的基本原理开始
2. 每一步都必须逻辑上从前一步推导
3. 记录你做出的新假设（要明确）
4. 将新的构建与原始概念进行比较

## 不同领域的常见基本原理

### 商业/经济学
- 人们的行为以自我利益为动机（有例外）
- 供需影响价格
- 价值 = 愿意支付的意愿
- 稀缺性创造价值

### 心理学/人类行为
- 人类寻求快乐，避免痛苦
- 社会地位对人类很重要
- 认知偏差存在（不是“人们是理性的”）
- 习惯通过重复形成

### 技术
- 信息可以以近乎零的成本复制
- 网络效应：价值 ∝ n²（梅特卡夫定律）
- 摩尔定律（晶体管密度每两年翻一番）
- 计算有能量成本

### 物理/现实
- 能量是守恒的
- 熵增加
- 光速是恒定的
- 因果关系先于效果

## 假冒的基本原理：红旗

注意伪装成基本原理的声明：
- “人们总是更喜欢X”（过度概括）
- “这只是人性”（通常是文化的）
- “经济学证明…”（许多经济学流派）
- “科学说…”（哪项研究？复制的？）

**测试：** 你能指出具体的证据/机制，还是它是民间智慧？

## 强有力的问题

- “为什么这是真的？”（重复5次 - “5个为什么”）
- “为了使这个有效，什么必须是真实的？”
- “如果我不能用这种方式做，还有什么其他途径？”
- “一个没有背景的异星人会怎么想？”
- “如果相反是真的怎么办？”

## 示例：分解“大学教育需求”

**假设：**
- 需要学位才能得到好工作
- 学习需要正式机构
- 18-22岁是正确的年龄
- 四年是最优持续时间

**基本原理：**
- 雇主想要有技能的工人
- 技能可以证明
- 学习需要时间和练习
- 知识可以传递

**重建：**
→ 雇主想要证明技能
→ 证明可以是：学位 OR 立档 OR 测试 OR 跟踪记录
→ 因此，学位是一个选择，而不是必需品
→ 见识：关注技能 + 证明，而不仅仅是证书

## 阅读中的实际应用

在分析内容时：
1. **识别主张**似乎绝对的
2. **挑战基础**：“这必须是真的吗？”
3. **验证因果关系**：“A真正导致B，还是只是相关？”
4. **仅从无可争议的事实重建论点**
5. **比较**保留的与假设的

这揭示了薄弱的论点并加强了有效的论点。
