# 域名头脑风暴者

此技能通过生成创意选项并检查哪些域名实际可注册，帮助您找到适合项目的完美域名。

## 何时使用此技能

- 开始新的项目或公司
- 发布产品或服务
- 创建个人品牌或作品集网站
- 重塑现有项目
- 注册副项目的域名
- 当您的首选域名已被占用时寻找可用替代方案

## 此技能的功能

1. **理解您的项目**：分析您正在构建的内容及其目标受众
2. **生成创意名称**：创建相关且易于记忆的域名选项
3. **检查可用性**：验证哪些域名在多个顶级域名中实际可用
4. **多个扩展名**：建议 .com、.io、.dev、.ai、.app 等
5. **提供替代方案**：如果顶级选择已被占用，则提供变体
6. **品牌洞察**：解释为什么某些名称效果良好

## 域名命名技巧

### 什么是一个好的域名

✓ **短**：理想情况下不超过 15 个字符
✓ **易于记忆**：易于回忆和拼写
✓ **易发音**：可以在对话中说出来
✓ **描述性**：暗示您所做的事情
✓ **可品牌化**：足够独特以脱颖而出
✓ **无连字符**：更容易口头分享

### TLD 指南

- **.com**：通用，受信任，非常适合企业
- **.io**：科技初创公司，开发者工具
- **.dev**：面向开发者的产品
- **.ai**：AI/ML 产品
- **.app**：移动或网络应用程序
- **.co**：.com 的替代品
- **.xyz**：现代、创意项目
- **.design**：创意/设计公司
- **.tech**：科技公司

## 工作流程示例

### 创业启动

1. 描述您的创业想法
2. 获取 10-15 个跨 TLD 的域名建议
3. 查看可用性和定价
4. 选择前三个最喜欢的
5. 立即注册

### 个人品牌

1. 分享您的姓名和职业
2. 获取变体（firstname.com、firstnamelastname.dev 等）
3. 检查社交媒体名称的可用性
4. 在平台上注册一致的品牌

### 产品命名

1. 描述产品及其目标市场
2. 获取创意、可品牌化的名称
3. 检查商标冲突
4. 核实域名和社交媒体的可用性
5. 用目标受众测试名称

## 成功技巧

1. **迅速行动**：好的域名很快就会被占用
2. **注册变体**：获取 .com 和 .io 以保护品牌
3. **避免数字**：难以口头传达
4. **检查社交媒体**：确保 @username 可用
5. **大声说出**：测试是否易于发音
6. **检查商标**：确保没有法律冲突
7. **考虑长远**：5 年后它是否仍然有意义？

## 定价背景

在建议域名时，我会注明：
- 标准域名：约 $10-15/年
- 高级 TLD（.io、.ai）：约 $30-50/年
- 已占用域名：如果列出，则市场价格
- 高级域名：数百到数千美元
