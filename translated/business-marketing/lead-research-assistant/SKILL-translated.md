# 领先研究助理

这项技能通过分析您的产品/服务，了解您的理想客户画像，并提供可操作的接触策略，帮助您识别和筛选潜在的商业线索。

## 何时使用此技能

- 为您的产品/服务寻找潜在客户或客户
- 建立要联系的公司名单以建立合作伙伴关系
- 识别销售接触的目标账户
- 研究符合您理想客户画像的公司
- 为业务发展活动做准备

## 此技能的功能

1. **理解您的业务**：分析您的产品/服务、价值主张和目标市场
2. **识别目标公司**：根据以下标准找到符合您理想客户画像的公司：
   - 行业和领域
   - 公司规模和位置
   - 技术堆栈和使用的工具
   - 成长阶段和资金
   - 您的产品解决的问题
3. **优先排序线索**：根据匹配分数和相关性对公司进行排名
4. **提供接触策略**：建议如何以个性化信息接触每个线索
5. **丰富数据**：收集有关决策者和公司背景的相关信息

## 如何使用

### 基本用法

只需描述您的产品/服务以及您在寻找什么：

```
我正在构建[产品描述]。在[位置/行业]找到10家适合此产品的公司。
```

### 与您的代码库一起使用

为了获得更好的结果，请从您的产品源代码目录运行以下命令：

```
查看我在这个存储库中构建的内容，并识别[位置/行业]中从该产品中受益的前10家公司。
```

### 高级用法

为了更精确的研究：

```
我的产品：[描述]
理想客户画像：

- 公司规模：[规模范围]
- 位置：[位置]

为我找到20个合格的线索，并为每个线索提供接触策略。
```

## 指令

当用户请求线索研究时：

1. **理解产品/服务**
   - 如果在代码目录中，分析代码库以了解产品
   - 就价值主张提出澄清问题
   - 确定关键功能和优势
   - 了解它解决的问题

2. **定义理想客户画像**
   - 确定目标行业和领域
   - 识别公司规模范围
   - 考虑地理偏好
   - 了解相关痛点
   - 记录任何技术要求

3. **研究和识别线索**
   - 搜索符合标准的公司
   - 寻找需求信号（职位发布、技术堆栈、最新新闻）
   - 考虑增长指标（资金、扩张、招聘）
   - 识别具有互补产品/服务的公司
   - 检查预算指标

4. **优先排序和评分**
   - 为每个线索创建匹配分数（1-10）
   - 考虑以下因素：
     - 与ICP的匹配度
     - 立即需求信号
     - 预算可用性
     - 竞争格局
     - 时间指标

5. **提供可操作的输出**

   对于每个线索，提供以下信息：
   - **公司名称**和网站
   - **为什么他们是合适的**：基于他们业务的特定原因
   - **优先分数**：1-10，并解释
   - **决策者**：要针对的角色/头衔（例如，“工程副总裁”）
   - **接触策略**：个性化的方法建议
   - **价值主张**：您的产品如何解决他们的具体问题
   - **对话开场白**：在接触中提及的具体点
   - **LinkedIn URL**：如果有的话，便于建立联系

6. **格式化输出**

   以清晰、可扫描的格式呈现结果：

   ```markdown
   # 线索研究结果
   
   ## 摘要
   - 总共找到的线索：[X]
   - 高优先级（8-10）：[X]
   - 中等优先级（5-7）：[X]
   - 平均匹配分数：[X]
   
   ---
   
   ## 线索 1：[公司名称]
   
   **网站**：[URL]
   **优先分数**：[X/10]
   **行业**：[行业]
   **规模**：[员工人数/收入范围]
   
   **为什么他们是合适的**：
   [基于他们业务的2-3个具体原因]
   
   **目标决策者**：[角色/头衔]
   **LinkedIn**：[URL如果有的话]
   
   **对他们来说的价值主张**：
   [针对该公司的具体好处]
   
   **接触策略**：
   [个性化的方法 - 提及特定的痛点、最近的公司新闻或相关背景]
   
   **对话开场白**：
   - [具体点 1]
   - [具体点 2]
   
   ---
   
   [为每个线索重复]
   ```

7. **提供下一步行动**
   - 建议将结果保存为CSV以导入CRM
   - 提供撰写个性化接触信息的建议
   - 根据时间推荐优先级
   - 建议对顶级线索进行更深入的研究
