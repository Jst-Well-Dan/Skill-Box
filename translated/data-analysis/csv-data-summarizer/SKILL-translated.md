### 自动分析步骤：

**技能通过首先检查数据，然后确定哪些分析最相关，智能地适应不同的数据类型和行业。**

1. **加载和检查** CSV文件到pandas DataFrame
2. **识别数据结构** - 列类型、日期列、数值列、类别
3. **根据数据中的内容确定相关分析**：
   - **销售/电子商务数据**（订单日期、收入、产品）：时间序列趋势、收入分析、产品性能
   - **客户数据**（人口统计、细分、地区）：分布分析、细分、地理模式
   - **财务数据**（交易、金额、日期）：趋势分析、统计摘要、相关性
   - **运营数据**（时间戳、指标、状态）：时间序列、性能指标、分布
   - **调查数据**（分类响应、评分）：频率分析、交叉表、分布
   - **通用表格数据**：根据找到的列类型进行适应

4. **只为特定数据集创建有意义的可视化**：
   - 仅当存在日期/时间戳列时才绘制时间序列图
   - 仅当存在多个数值列时才绘制相关性热图
   - 仅当存在分类列时才绘制类别分布
   - 当相关时为数值分布绘制直方图
   
5. **自动生成全面输出**，包括：
   - 数据概述（行、列、类型）
   - 与数据类型相关的关键统计和指标
   - 缺失数据分析
   - 多个相关可视化（仅适用于适用的情况）
   - 基于在此特定数据集中发现的模式的可操作洞察
   
6. **在一个完整分析中展示一切** - 没有后续问题

**示例适应：**
- 医疗保健数据与患者ID → 重点关注人口统计、治疗模式、时间趋势
- 库存数据与库存水平 → 重点关注数量分布、再订购模式、SKU分析  
- 网络分析数据与时间戳 → 重点关注流量模式、转化指标、一天中的时间分析
- 调查响应 → 重点关注响应分布、人口统计分解、情感模式
